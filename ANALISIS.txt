1. DESCRIPCIÓN GENERAL DEL PROYECTO
================================================================================
Este es un proyecto educativo que implementa un sistema de gestión de recursos 
humanos completamente en JavaScript vanilla (sin frameworks). La aplicación se 
ejecuta en la consola del navegador y permite gestionar empleados, realizar 
operaciones matemáticas y calcular estadísticas.

El proyecto aborda conceptos fundamentales de JavaScript como variables, 
condicionales, arreglos, ciclos, funciones, objetos y métodos.

3. COMPONENTES PRINCIPALES
================================================================================

3.1 ARCHIVO: index.html
--------------------
- Página HTML básica y minimalista
- Meta tags: UTF-8 y viewport responsive
- Carga el script.js de forma diferida (defer)
- Sin estilos CSS ni contenido HTML evidente
- Todo el funcionamiento ocurre en la consola del navegador


3.2 ARCHIVO: script.js
--------------------
El archivo script.js está dividido en 3 secciones temáticas:

SECCIÓN 1: BIENVENIDA Y EVENTOS INICIALES
- Mensaje coloreado de bienvenida en consola
- Solicita el nombre del usuario mediante prompt()
- Greeting personalizado con alert()

SECCIÓN 2: OPERACIONES MATEMÁTICAS CON VALIDACIÓN
- Entrada de dos números desde el usuario
- Validación con isNaN() para números válidos
- Operaciones: suma, resta, multiplicación, división
- Funciones individuales usando arrow functions:
  * sumar(a, b) → a + b
  * restar(a, b) → a - b
  * multiplicar(a, b) → a * b
  * dividir(a, b) → validación para evitar división por cero
- Switch case para seleccionar operación
- Manejo de errores y normalización de entrada (.toLowerCase(), .trim())

SECCIÓN 3: GESTIÓN DE EMPLEADOS
- Base de datos local con 6 empleados predefinidos
- Cada empleado tiene propiedades: nombre, experiencia, salario, departamento
- Métodos dinámicos agregados a cada objeto:
  * getNivelExperiencia() → Clasifica en Junior/Mid/Senior/Expert
  * calcularBono() → 10% del salario


4. FUNCIONALIDADES IMPLEMENTADAS
================================================================================

4.1 FUNCIONES MATEMÁTICAS
--------------------------
- sumar(a, b): suma dos números
- restar(a, b): resta dos números
- multiplicar(a, b): multiplica dos números
- dividir(a, b): divide con validación de cero


4.2 FUNCIONES DE CÁLCULO ESTADÍSTICO
-----------------------------------
- calcularPromedioExperiencia(): Promedio de años usando bucle for
- calcularSalarioPromedio(): Promedio de salarios usando reduce()
- Retornan valores con 2 decimales (.toFixed(2))


4.3 FUNCIONES DE GESTIÓN DE EMPLEADOS
-----------------------------------
- agregarEmpleado(nombre, experiencia, salario, departamento):
  * Valida todos los datos de entrada
  * Crea nuevo objeto con métodos
  * Actualiza estadísticas en consola
  
- filtrarPorExperiencia(minExperiencia):
  * Usa forEach para iterar
  * Devuelve empleados que cumplan el criterio
  * Muestra información formateada

- mostrarEmpleados():
  * Usa map para transformar datos
  * Muestra información completa de cada empleado
  * Incluye nivel, bono y departamento


4.4 MENÚ INTERACTIVO
-------------------
- iniciarAplicacion(): Bucle while que mantiene activo el menú
- Opciones:
  1. Agregar empleado
  2. Filtrar por experiencia
  3. Mostrar todos los empleados
  4. Calcular promedios
  5. Salir

================================================================================

2. Script se ejecuta:
   - Muestra mensaje de bienvenida coloreado

   - Muestra mensaje de bienvenida personalizado (alert)
